# Лендинг услуг техспеца

Одностраничный сайт-визитка Катерины Степановой — технического специалиста и графического дизайнера. Помогаю онлайн-школам и экспертам с автоматизацией: создание сайтов (код и Tilda), Salebot, AmoCRM, интеграции, Telegram-боты.

## Стек

- HTML, CSS, JavaScript — без сборщиков и фреймворков
- Весь сайт в одном файле `index.html`

## Как запустить

Открыть `index.html` в браузере. Никаких `npm install`, dev-серверов и сборки.
